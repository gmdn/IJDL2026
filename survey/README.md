# Survey Data

This folder contains the anonymized data collected during the preliminary user evaluation of **Annotaterm**, as reported in the accompanying IJDL paper.

## Description

The evaluation involved **14 participants** enrolled in a Master's Degree course in *Modern Languages for Communication and International Cooperation*. Participants completed an annotation task using Annotaterm and subsequently answered a short questionnaire.

The questionnaire consisted of:

- background questions on previous annotation experience;
- task completion status;
- six 5-point Likert-scale questions evaluating usability and suitability;
- two open-ended questions collecting qualitative feedback.

## Files

### `questions.csv`

Contains the questionnaire structure.

Columns include:

- **Question** – question identifier (Q1–Q10)
- **Text** – questionnaire text
- **Responses** – expected response type or possible values

### `answers.csv`

Contains the anonymized responses collected during the evaluation.

Each row corresponds to one participant.

No personally identifiable information is included.

## Privacy

The dataset has been fully anonymized before publication.

Participants did not provide names, email addresses, or any other identifying information.
